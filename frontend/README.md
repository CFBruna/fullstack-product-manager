# Frontend - Product Manager

Frontend application for the Product Management System. Built with React, TypeScript, and Vite.

## Tech Stack
- **Framework:** React + TypeScript + Vite
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios
- **Routing:** React Router DOM
- **Validation:** Zod
- **Icons:** Lucide React

## Getting Started

### Prerequisites
- Node.js installed
- Backend running on port 3001 (for API integration)

### Installation
Navigate to the frontend directory:
```bash
cd frontend
```

Install dependencies:
```bash
npm install
```

### Running the App
Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.
